                                                  Abstract

            Multi-objective optimization underlies many machine learning problems, from
            multi-task learning to LLM safety/alignment. A standard approach scalarizes the
            objectives via a weight vector in the unit simplex, but in practice the right weights
            are rarely known, motivating methods that approximate solutions uniformly across
            all possible weights. Existing approaches either return a single Pareto-stationary
            point or discretize the simplex and solve each scalarized problem independently,
            discarding valuable information that could be reused across weights.
            We introduce a first-order bundle method that systematically reuses zeroth- and
            first-order information across weight vectors. The algorithm maintains a bundle
            of previously computed gradients and function values, from which it constructs
            a progress criterion certifying solution quality uniformly over the simplex, and
            adaptively refines the bundle where progress is largest. We establish iteration and
            oracle complexity bounds under non-convexity — the regime that directly capture
            common machine learning training settings. We further demonstrate the advantage
            of our method over the uniform discretization method as the baseline on LLM
            safety/alignment benchmarks.


1    Introduction

Many problems in machine learning involve the simultaneous optimization of multiple objective
functions. In physics-informed neural networks (PINNs) training, multiple loss terms induced by
PDE residuals and boundary or initial conditions must be optimized simultaneously. In multi-task
learning, a shared model must perform well across several tasks at once, each with its own loss
function [R97]. In multi-class classification, the per-class losses define a collection of objectives that
must be jointly minimized to achieve good predictive performance across all classes [Bis06]. These
settings—and others such as hyperparameter optimization, federated learning, and fairness-aware
training—give rise to the multi-objective optimization (MOO) problem.

forms the weighted combination Fλ (x) := k=1 λk Fk (x) for a weight vector λ in the unit simplex

well-understood scalar optimization problem. However, in many settings, such as multi-task learning,
the weights need to be assigned manually at prior, and the overall performance depends heavily
on the assigned weights. One potential resolution is to find good solutions for all weight vectors
λ ∈ ∆K —that is, to approximate the entire Pareto front. This raises the true challenge of MOO,
requiring solving a continuum of optimization problems, one for each λ, which is computationally
prohibitive.
In this paper, we introduce a first-order bundle method for smooth multi-objective optimization that
addresses this gap. The central idea is to maintain a bundle—a collection of previously computed
zeroth-order and first-order information at a set of iterates—and to construct from this bundle a
progress criterion that certifies solution quality uniformly over all λ ∈ ∆K .

1.1   Our Methods and Contributions
      1. We introduce a general algorithmic framework (Section 4) for bundle-based smooth MOO,
         which iteratively selects the weight vector λ with the largest progress criterion and refines the
         bundle at that weight. We present both algorithm iteration complexity and oracle complexity
         results (Theorem ??).
      2. We provide a concrete instantiation of the framework under smoothness and various convex-
         ity assumptions (Example 1,2), deriving closed-form expressions for the progress criteria
         that are efficiently computable from the bundle.
      3. We demonstrate the effectiveness of our method on multi-class logistic regression and
         multi-class cross-entropy loss minimization, as well as multi-task learning benchmarks,
         showing that the bundle method efficiently traces the Pareto front while reusing gradient
         information across weight vectors.

1.2   Related Work

Existing approaches address this challenge from different angles. Methods such as the Multiple
Gradient Descent Algorithm (MGDA) [Dés12] and its variants [SK18, LLJ+ 21, NSA+ 22] seek a
single Pareto-stationary point by combining task gradients, but they do not characterize the full
Pareto front. Grid-based scalarization methods [SK18] discretize ∆K and solve each subproblem
independently, but this is wasteful: gradient information computed at one weight vector is discarded
when solving for another. Neither approach provides a principled mechanism for reusing zeroth-order
and first-order information across different scalarization weights.

One design choice in Algorithm 1 merits discussion.

Warm-starting. On the first pass through the grid, each new grid point’s gradient descent is
initialised from the previous grid point’s current iterate, exploiting the continuity of the optimal
solution map λ 7→ x∗ (λ) (which is automatic under strong convexity). After the first pass, each
grid point continues from its own current iterate, since each xb(g) is now closer to x∗ (λ(g) ) than its
neighbour’s iterate would be. The lexicographic enumeration ensures consecutive grid points are
ℓ1 -close, which makes the first-pass warm-starting effective.

error) tuples at periodic checkpoints, triggered when M gradient evaluations have accumulated since
the last checkpoint. In both algorithms, the actual checkpoint occurs at the next natural boundary (a
pass-completion for the baseline; an outer iteration for Algorithm 3) after the threshold is reached.

6   Conclusions and future directions
In this paper, we have developed a first-order bundle method for solving the smooth multi-objective
optimization problem, which has broad applications in machine learning, such as multi-task learning,
neural architecture search, hyperparameter optimization, generative modeling, transfer and domain
adaptation, recommender systems, reinforcement learning with smooth rewards, federated learning
with smooth local loss. Some future directions include extending the setting to non-smooth multi-
objective optimization, which has applications in fairness-accuracy tradeoff and robust and adversarial
learning.
